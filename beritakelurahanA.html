Rumah Warga Sukoharjo Terbakar, 2 Sapi dan 1 Motor Hangus

Rumah warga di Tegalrejo RT 003/RW 008, Desa Dukuh, Kecamatan Sukoharjo, Kabupaten Sukoharjo terbakar pada Selasa (6/6/2023) dini hari. Dua sapi, satu sepeda motor dan tiga sepeda onthel ludes dilalap api.

Solopos.com, SUKOHARJO — Rumah warga di Tegalrejo RT 003/RW 008, Desa Dukuh, Kecamatan Sukoharjo, Kabupaten Sukoharjo terbakar pada Selasa (6/6/2023) dini hari. Sumber api diduga berasal dari sisa pembakaran jerami pakan ternak.

Camat Sukoharjo, Havid Danang Purnomo Widodo, mengatakan rumah yang terbakar itu milik Sriyono, 55. Musibah itu terjadi sekitar pukul 01.30 WIB. Api membakar dua sapi, satu unit sepeda motor Honda Supra, pompa air, televisi, speaker, dan tiga sepeda onthel milik korban. Sriyono sendiri mengalami luka bakar sampai harus dilarikan ke rumah sakit.

Warga sempat membantu memadamkan api dengan peralatan seadanya sebelum meminta bantuan pemadam kebakaran. “Pemerintah Kecamatan Sukoharjo telah menyerahkan bantuan makanan dan uang tunai untuk korban,” terang Havid.

Sementara itu, Kepala Bidang Pemadam Kebakaran (Kabid Damkar) Satpol PP Sukoharjo, Margono, mengatakan tetangga depan rumah korban sempat melihat api berkobar dis ekitar rumah korban. Tetangga itu kemudian memberitahu pemilik rumah dan selanjutnya menelepon Damkar.
